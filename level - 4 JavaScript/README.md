1. What does javascript do?

Javascript makes a website interactive this language creates an interaction
between the user and the website. Javascript makes the website more dynamic
and the actions of the user determines the course of the website.

2. Difference between HTML CSS JS

HTML or hypertext markup language is the base for the website. With this
language we create the base of our website and it is a structure
CSS or cascading style sheets adds style to the webpage and changes the 
color, font , size etc
Javascript makes the website interactive and here there is a form of 
communication between the user and the webpage.
